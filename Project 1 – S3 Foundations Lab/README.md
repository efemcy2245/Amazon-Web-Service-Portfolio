# Project 1 – S3 Foundations Lab

## Overview

This project is the first hands-on lab in my Amazon S3 portfolio.  
The goal is to build a solid understanding of core S3 concepts and operations by creating and organizing a bucket, uploading sample objects, applying metadata and tags, and using AWS CLI to perform basic object management tasks.

This lab focuses on:
- bucket creation
- object organization through prefixes
- upload and download operations
- object copy and delete operations
- user-defined metadata
- object tagging
- basic AWS CLI scripting

---

## Objectives

The main objectives of this project are:

- create an S3 bucket with a clean and unique naming convention
- organize objects using logical prefixes
- upload at least 20 sample files of different types
- apply metadata to selected objects
- apply tags to selected objects
- verify object metadata and tagging
- use AWS CLI for list, copy, download, and delete operations
- document the work in a clean GitHub repository
