# Project 01 - AWS S3 Foundations

This project documents my first hands-on lab with Amazon S3.

## Goal
Create a private S3 bucket, upload sample files, organize objects with prefixes, and test basic CLI operations.

## Topics covered
- Buckets and objects
- Prefixes
- Metadata
- Tags
- Upload and download
- Copy and delete
